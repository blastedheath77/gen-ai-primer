The Prediction Game

What AI actually does

A large language model does one thing at its core: given a sequence of words, it predicts what comes next. Conversations, essays, and code all emerge from repeating this single act.

Key Points

  • The model isn't "thinking" — it's calculating statistical likelihood
  • Even you can predict common phrases, which is what the model scales up enormously
  • Ambiguous sentences reveal that multiple valid continuations exist

But how does the model even read a sentence? It doesn't see words the way you do.
