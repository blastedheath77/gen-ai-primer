Probability, Not Certainty

A landscape of possibilities, not a single right answer.

The model doesn't pick one "right answer." It calculates a probability for every possible next token — thousands of candidates, each with a score. The top few are all plausible; the choice between them is where randomness and creativity live.

Key concepts

Fresh distribution each step no memory beyond text
No look-ahead one token at a time
Top vs. surprising creativity dial
Accuracy vs. creativity core trade-off
