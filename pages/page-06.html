Context Changes Everything

The same word, different meanings

A word in isolation is ambiguous. An LLM doesn't look up a fixed meaning — it recalculates meaning from the surrounding context. "Bank" near "river" points to one location in meaning-space; "bank" near "money" points somewhere entirely different.

Key Points

  • Static word embeddings are a starting point; real LLMs use contextual embeddings
  • This is what makes LLMs far more capable than simple keyword matching
  • The same mechanism explains why word order matters
  • This contextual recalculation happens for every token at every layer
