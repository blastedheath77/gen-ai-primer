Beyond Text

The same prediction trick — applied to images, audio, and code.

You've seen how a transformer predicts the next word. The same mechanism works on anything that can be broken into a sequence. Images, audio, video, code — all become token streams. The architecture is domain-agnostic.

Key concepts

Universal tokenisation any data → sequences → transformer
One mechanism, many domains images, audio, code — same engine
Multimodal = one stream text and image tokens interleaved
Cross-domain transfer progress in one domain speeds up others
